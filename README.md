# fukkit-repo
Repository of Fukkit artifacts

## Usage
```groovy
repository {
    name = 'FukkitMC'
    url = 'https://raw.githubusercontent.com/FukkitMC/fukkit-repo/master'
}
```
```kotlin
repository {
    name = "FukkitMC"
    url = uri("https://raw.githubusercontent.com/FukkitMC/fukkit-repo/master")
}
```
